# Chess-
Play Chess on the Terminal. that's it.

Use the Arrow keys to select/move pieces.

Q - Exit game.
ENTER - Select Area.
Up - move cursor up.
Down - move cursor down.
Right - move cursor to the right.
Left - move cursor to the left.
